Nginx jest narażony na dostęp do Internetu na porcie 80. Gdy żądanie jest wysyłane do dowolnego punktu końcowego frontendu, frontend Express server obsługuje statyczne strony html. Gdy żądanie jest wysyłane ze strony internetowej gs_calc (lub z dowolnego /api endpoint) serwer sieciowy api express obsługuje te żądania.
